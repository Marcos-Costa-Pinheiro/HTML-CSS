Outras formatações

Código fonte/Pré formatação

A tag code utiliza a formatação de letras com o mesmo espaçamento.

A tagpre utiliza a pre formatação, ele leva em conta os espaços e quebras de linhas pre definidos.

Exemplo:


    num = int(input('Digite um número: ''))
    if num % 2 == 0:
        print(f'O número {num} é par')
    else
        print(f'O número {num} é ímpar')
